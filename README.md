# MERN Stack Project

## Introduction

> A shopping list application created with the MERN Stack

## Code Samples

```javascript
function App() {
  return (
    <Provider store={store}>
      <div className="App">
        <AppNavbar />
        <ShoppingList />
      </div>
    </Provider>
  );
}
```

## Installation

> run ```npm install``` and then ```npm run server``` to start the backend and frontend servers.
